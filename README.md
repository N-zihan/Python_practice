# 我的 Python 入门练习项目 🐍

> 一个记录了我初学 Python 时所有练习的仓库。

## 📝 项目简介

这个仓库是我高一自学 Python 的第一站，包含了从基础的 `print` 语句，到有完整交互界面的计算器程序。它见证了我的学习过程，也是我的第一个实践项目。

## ✨ 包含内容

*   `while计算器.py`: 一个带错误处理和猫娘风格的交互式平均值计算器。
*   `练习：class.py`: 关于面向对象编程（OOP）的初步尝试。
*   `你能否偷到手机.py`: 一个脑洞大开的 `if-else` 条件判断练习。
*   ...以及其他各种语法练习文件。

## 🚀 如何运行

1.  确保你的电脑安装了 Python 3.x 版本。
2.  克隆或下载本仓库到本地。
3.  在终端中进入项目文件夹，运行 `python 文件名.py` 即可，例如：
    ```bash
    python "while计算器.py"
    ```
